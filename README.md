# CTF-BOX

Based on Ubuntu/bionic(18.04)

## Setup

```
$ vagrant up
$ vagrant ssh
```

## Tools

### Pwn

| name | info | link |
| -------- | -------- | -------- |
| readelf | Show information of ELF files ||
| objdump | Show information of object files ||
| gdb | Gnu DeBugger | https://www.gnu.org/software/gdb/ |
